## 💡 记录标准
1. **时间戳** - 每个记录都要有明确的时间戳
2. **详细描述** - 记录要详细，便于后续查阅和理解
3. **分类清晰** - 按照类型和重要性进行分类
4. **可追溯** - 修改要能够追溯到原因和过程
5. **持续更新** - 定期更新和维护记录内容
