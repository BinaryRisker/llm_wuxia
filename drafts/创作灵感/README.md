## 💡 记录原则
1. **及时记录** - 灵感稍纵即逝，要及时记录下来
2. **分类整理** - 按照不同类型进行分类管理
3. **定期回顾** - 定期回顾旧的灵感，可能会有新的启发
4. **实用导向** - 记录要考虑实际应用的可能性
5. **创新优先** - 优先记录独特和创新的想法
